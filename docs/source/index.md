# AIND Zarr Utils

Utilities for working with ZARR data and AIND metadata

## Installation

```bash
pip install aind-zarr-utils
```

For development:
```bash
git clone https://github.com/AllenNeuralDynamics/aind-zarr-utils.git
cd aind-zarr-utils
uv sync
```

## Quick Start

```python
import aind_zarr_utils

# Your code here
```
